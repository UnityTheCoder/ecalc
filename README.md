## Getting Started

### Installation

```bash
sudo sh -c "curl -L https://raw.githubusercontent.com/UnityTheCoder/ecalc/main/ecalc > /usr/bin/ecalc; chmod +x /usr/bin/ecalc"
```

## Usage
```bash
ecalc 4+5
ecalc 'log(3)'
ecalc 4^3     (auto translation to) ~>   pow(4,3)
ecalc exp(4)
ecalc 'asin(0.55)'
ecalc 'inv(5)'        =>           -5
ecalc 'cbrt(8)'                     ~>   num ** (1 / 3)
```


### Constants

- `e`   ~> Euler 
- `pi`  ~> Pi
- `tau` ~> Tau
- `inf` ~> infinite
- `nan` ~> NaN
